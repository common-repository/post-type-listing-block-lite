This plugin can be used for all type of Gutenberg Posts and Pages. The main aim of this plugin is to make the blog section of the website most user-friendly and convenient to read.

== Description ==

This plugin allows you to set custom post listing. You can Categorize the different blog posts, choose the number of posts you want to display per page, make grid wise blog posts after assigning several columns per grid.

In addition to that, this Plugin allows you to display different parameters related to blog posts. I.e. : Post Content, Post Title, Feature Image, Author Name, Date, Comment Count, Post Box Design. You can decide whether you want to display full post or Excerpt. Further, this plugin allows you to Customize Title and Feature Image. You can make the block design as per your choice by selecting a different margin.

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of Post type listing block lite, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type "Post type listing block lite." then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.
